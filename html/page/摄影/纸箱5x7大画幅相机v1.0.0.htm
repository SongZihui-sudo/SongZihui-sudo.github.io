最近在某宝闲逛时发现个神奇玩意儿——医用X光片！才八十多块钱就能买到一盒5x7英寸的，比普通摄影胶片便宜很多。赶紧翻墙查了下资料，原来这种感蓝胶片真的能当摄影道具用！不过要提醒各位的是，这可不是普通的底片，大家收到后记得先在暗房里试试水，别直接拿去拍人像哦。

冲洗

这种胶片按照包装上的说明是可以在安全灯下打开的，因为我没有暗房，所以上片，冲洗都在暗袋里进行。

因为是在暗袋里进行冲洗操作，所以要省略了一切需要倒水，倒药水的操作。最终在冲洗的过程中只保留显影和定影这两步操作，因为在暗袋里没法倒药水，遂选用了巨星的牙科x光片显影定影二合一套药(20元上下250ml)。

显影方式为盘显，用了一个不锈钢饭盒装药水。

重点来了👉 原装的药水药力非常大。 如果用原液直接冲洗瞬间底片变得全黑。所以在后面冲洗的时候使用了 1:4 的稀释液。 经多次实验， 选择冲洗时间为 30s 上下 。目前来看这样得到的底片反差较低。

拍摄

由于这种胶片是感蓝的，所以感光度相比于感绿胶片低一点，在拍摄时按照 iso 50 曝光。

用小孔的直径和焦距可以计算出光圈，这样就可以直接进行测光。 用秒表倒计时控制曝光时间。

纸盒相机


相机前

为了方便拍摄，我制作了一个纸盒相机。

最近尝试用快递盒DIY了一款简易X光片摄影装置。材料非常简单：取一个废弃的快递盒，裁切出能容纳5x7英寸胶片的开口作为片夹主体。考虑到胶片在操作中容易被刮花，我在背板位置用硬纸板加固，并特意贴了一张报废的X光片作为保护层——这种自带磨砂质感的废片意外地起到了缓冲作用。 固定胶片时发现双面胶比预想中更实用：将胶条沿背板边缘贴出定位框，既能让底片精准卡位，又不会在频繁更换时残留胶渍。 这个零成本的方案虽然简陋，但意外地解决了暗袋操作时反复取放胶片的困扰。


相机内

机身内部上做出一个框来定位底片，最终确定焦距为 110mm，通过在网上资料查询，最佳孔径在 0.4mm 左右，选择了一个 0.4mm 小孔片。

内壁尽量使用黑色材料遮光，底片反差低的一个原因也有可能是在这，内壁消光效果不好。
